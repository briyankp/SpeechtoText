# SpeechtoText

Use OpenAI's Whisper APIs to transcribe and convert speech or audio into text. 
Whisper APIs are really great and it outperforms most other paid speech to text tools.

- [ ] Go to Google Drive: https://drive.google.com and search colaboratory
- [ ] Install and configure Google Colaboratory 
- [ ] Install Whisper AI - Command in the file
- [ ] Upload audio or video - Just drag the Mp3 or mp4 file
- [ ] Run Whisper AI - Command in the file
- [ ] Review results and check the new file created with the text and timestamps
